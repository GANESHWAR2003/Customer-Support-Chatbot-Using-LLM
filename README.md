# Customer-Support-Chatbot-Using-LLM
An AI-powered customer support chatbot using Large Language Models (LLMs) that understands user queries and provides accurate, context-aware responses. It can process documents like PDFs and deliver real-time conversational support, improving efficiency and reducing manual workload.
